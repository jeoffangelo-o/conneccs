# ConneCCS Logo

Place your ConneCCS logo image in this directory as `logo.png`.

## Requirements:
- **File name**: `logo.png`
- **Recommended size**: 34x34 pixels (or larger, will be scaled)
- **Format**: PNG with transparent background preferred
- The logo will appear in the sidebar and as the favicon

## Fallback:
If no logo.png is found, the system will display the default "C²" text logo.

## Design Guidelines:
Based on the ConneCCS branding:
- Use colors that complement the system theme (blues, purples)
- Keep it simple and recognizable at small sizes
- Ensure good contrast for visibility
- Consider the dark theme background (#141720)
