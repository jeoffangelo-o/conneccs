# Quick Guide: Role-Based Tagging

## 🎯 How to Mention/Tag Roles

### Step-by-Step:

1. **Go to Messages** (click "Messages" in sidebar)

2. **Select a channel** (e.g., #general)

3. **Type your message with `@` symbol:**
   ```
   @faculty Please submit your reports by Friday
   ```

4. **Press Enter** to send

5. **Result:** All faculty members will see this message highlighted!

## 📝 Available Tags

Type these in your messages:

- `@dean` - Notify Dean's Office
- `@faculty` - Notify all faculty members  
- `@program_chair` - Notify program chairs
- `@research_coordinator` - Notify research coordinators

## 💡 Examples

### Example 1: Simple Mention
**Type:**
```
@faculty Meeting tomorrow at 10 AM
```

**What happens:**
- Message sends to #general
- All faculty see it highlighted with blue border
- Others see it as normal message

### Example 2: Multiple Mentions
**Type:**
```
@dean @program_chair Need approval for budget
```

**What happens:**
- Both Dean and Program Chairs see it highlighted
- Easy to get multiple roles' attention

### Example 3: Mention in Sentence
**Type:**
```
Hey everyone, @faculty please check your email!
```

**What happens:**
- Works anywhere in the message
- Faculty members see it highlighted

## 🔥 Pro Tips

✅ Use `@faculty` for department-wide announcements
✅ Use `@program_chair` for program-specific matters
✅ Use `@dean` for urgent approvals
✅ Combine tags: `@dean @faculty` for multiple groups

❌ Don't overuse tags (causes notification fatigue)
❌ Don't tag irrelevant roles

## 🚀 Autocomplete Feature

When you type `@`, suggestions appear:

1. Type `@` in message box
2. See dropdown with role options
3. Click a role or keep typing
4. Role is inserted automatically
